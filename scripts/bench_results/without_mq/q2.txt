## Installing langchain-anthropic

```bash
pip install langchain-anthropic
```

## Using langchain-anthropic

The package provides the following main exports:

- **`ChatAnthropic`** - The main chat model class for interacting with Claude models
- **`AnthropicLLM`** - Legacy LLM interface
- **`convert_to_anthropic_tool`** - Utility to convert LangChain tools to Anthropic format

### Basic Usage Example

```python
from langchain_anthropic import ChatAnthropic

# Initialize the model
llm = ChatAnthropic(model_name="claude-3-5-haiku-20241022")

# Simple invocation
response = llm.invoke("Hello, how are you?")

# Streaming
for token in llm.stream("Tell me a story"):
    print(token.content, end="")

# Async streaming
async for token in llm.astream("Tell me a story"):
    print(token.content, end="")
```

### Environment Setup

You'll need to set your Anthropic API key:
```bash
export ANTHROPIC_API_KEY="your-api-key"
```
