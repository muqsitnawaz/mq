Based on the documentation, here's how to install and use the langchain-anthropic partner package:

## Installation

```bash
pip install langchain-anthropic
```

## What it provides

The `langchain-anthropic` package contains the LangChain integration for Anthropic's generative models (Claude).

## Documentation

- **API Reference**: https://reference.langchain.com/python/integrations/langchain_anthropic/
- **Conceptual guides, tutorials & examples**: https://docs.langchain.com/oss/python/integrations/providers/anthropic

## Basic Usage

After installation, you can import and use the ChatAnthropic class:

```python
from langchain_anthropic import ChatAnthropic

# Create a chat model instance
llm = ChatAnthropic(model="claude-3-opus-20240229")

# Use it in your LangChain chains and agents
response = llm.invoke("Hello, how are you?")
```
